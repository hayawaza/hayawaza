プライバシーポリシー

Hayawaza（以下「本アプリ」）のプライバシーポリシーをご説明します。

1. 収集する情報

本アプリは、いかなる個人情報・利用データも収集しません。

  • キーボード入力の内容は一切記録・送信しません
  • 使用履歴・統計情報の収集は行いません
  • 外部サーバーへの通信は行いません

2. キーボード操作の取り扱い

本アプリはショートカットキー表示のため、特定のキー（呼び出しキーとして設定したキー）の押下状態を検出します。検出した情報はオーバーレイ表示の制御にのみ使用し、外部への送信・ファイルへの保存は行いません。

3. 設定データの保存

ユーザーが設定した表示オプション（表示位置・透過率・ホットキー等）は、お使いのPC内（ローカル）にのみ保存されます。クラウドや外部サービスへの送信は行いません。

4. 第三者への提供

本アプリは、いかなる第三者にも情報を提供しません。

5. お問い合わせ

本ポリシーに関するご質問は、Microsoft Store のサポートページよりお問い合わせください。

6. 変更について

本ポリシーは予告なく変更される場合があります。変更後の内容はこのページに掲載します。
